# MATLAB → C 定点 BM_module (ERB Band Merging) 转换经验

## 核心强制约束

1. 输入 `x`：**s32f20**，257 bins (log_gen 输出)
2. 权重 `erb_erb_fc_weight`：**u16f15**，`[192×64]` col-major（已从 .mat 提取为 C 数组）
3. 输出 `y`：**s32f20**，129 bins
4. 低频 bins 0-64：**直通**（bit-exact memcpy）
5. 高频 bins 65-256：**矩阵乘法** → bins 65-128

## 1. 输入对齐检查清单

| 参数 | MATLAB (`BM_module.m`) | C (`bm_fixed`) | 影响 |
|------|----------------------|----------------|------|
| 输入 bins | 257 (log_gen 输出 Q20) | 257 int32 Q20 | 一致 |
| 输出 bins | 129 | 129 | 一致 |
| 低频直通 | `y(1:65) = x(1:65)` | `memcpy(y, x, 65*4)` | **bit-exact** |
| 高频合并 | `round(x(66:end) * W * 2^(-15))` | Q20×Q15 → >>15 → saturate Q20 | **124 dB** |
| 权重格式 | u16f15, [192×64] col-major | `erb_erb_fc_weight[12288]` | 逐值一致 |
| 累加器 | MATLAB double → round | int64 → round(>>15) | ±0.5 LSB/项 |

## 2. 定点 Q 格式约定

```
信号路径:  log_gen Q20 [257] → BM → Q20 [129]
                           ↓
           bins 0-64:  直通 (Q20 → Q20, 0 LSB error)
           bins 65-256: 矩阵乘 [192] × [192×64] → [64]
               Q20 × u16f15 → Q35 → round(>>15) → Q20
               累加 192 项, int64 安全 (max ≈ 192×2³¹)
```

| 误差源 | 贡献 | 说明 |
|--------|------|------|
| 矩阵乘取整 >>15 | ±0.5 LSB/项 | 192 项累加 RMS ≈ 0.5×√192 ≈ 7 LSB 理论值 |
| 累加顺序差异 | 可忽略 | double (53-bit) vs int64 (63-bit) |
| 权重量化 | 0 | 直接用 MATLAB Q15 值，无额外量化 |

## 5. 代码模板

```c
void bm_fixed(const int32_t *x, const uint16_t *weight, int W_in, int W_out, int32_t *y) {
    int low = 65;
    memcpy(y, x, low * sizeof(int32_t));  // passthrough, bit-exact

    int hi_in = W_in - low;   // 192
    int hi_out = W_out - low; // 64
    int64_t r = 16384;  // 2^14 for >> 15

    for (int o = 0; o < hi_out; o++) {
        int64_t acc = 0;
        for (int i = 0; i < hi_in; i++) {
            int64_t prod = (int64_t)x[low + i] * weight[i + hi_in * o];
            if (prod >= 0) acc += (prod + r) >> 15;
            else           acc += (prod - r) >> 15;
        }
        y[low + o] = (acc > INT32_MAX) ? INT32_MAX :
                     (acc < INT32_MIN) ? INT32_MIN : (int32_t)acc;
    }
}
```
